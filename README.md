# InfoShield

A media literacy website for Australians to learn about misinformation.
Also, thanks to SahasTechnologies (Sahas Shimpi) for connecting my repository to vercel because my account was encountering some troubles.

## Setup

1. Make sure you have Node.js installed (nodejs.org)
2. Open a terminal in this folder
3. Run: `npm install`
4. Run: `npm run dev`
5. Open your browser at: http://localhost:5173

## Pages

- **Home** — Hero, stats, why it matters, video
- **The Issue** — Charts, political bias slider, key takeaways
- **Quiz** — 10-question media literacy quiz
- **How to Help** — Tips and fact-checking resources
- **Contact** — Contact form
